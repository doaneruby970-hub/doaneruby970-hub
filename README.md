## Good Fit

I am a good fit for focused projects such as:

- fixing or extending an existing scraper;
- turning a manual browser/data workflow into a repeatable script;
- collecting product, pricing, inventory, or market research data;
- building a small API around an automation workflow;
- integrating forms, CRMs, spreadsheets, email, or third-party APIs;
- creating a clean first MVP with documentation and handoff notes.

## Working Style

- Start with a small working version first.
- Keep outputs easy to inspect.
- Prefer simple, maintainable code over unnecessary platform complexity.
- Document setup, assumptions, and failure cases.
- Be clear about what is proven experience and what is comparable experience.
